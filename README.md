# Snapshop for VSCode

<p align="center"><img width="128" alt="" src="https://github.com/AuTsing/snapshop-extension/blob/main/assets/logo_128_128.png?raw=true"/></p>

Snapshop 是一款现代、快速、跨平台的 Web 图色处理应用。

Snapshop for VSCode 是 Snapshop 的 VSCode 实现。

## 特性

- ✨ 使用 Vue3 框架
- ✨ 使用 Antdv4 UI
- ✨ 纯 Web 实现，可以轻松移植至任何 Web 环境
- ⚡ 从 HTTP(S)，WS 协议加载图片
- ⚡ 图片预览
- ⚡ 图片取色
- ⚡ 图片取范围
- ⚡ 图片对比
- ⚡ 使用模板生成代码
- ⚡ 字库生成

## 使用

- 使用 **<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>** 调出命令面板，输入 `Snapshop` 打开应用。

**Enjoy!**
