### Snapshop v3.0.0 更新日志

-   🎈 更新 README
-   🛠 更新底层依赖
-   🐞 修复连续打开相同图片无效的问题
-   🐞 修复一些可能导致内存泄漏的问题

### 📌 2.x -> 3.x 迁移指南

-   原 `模板n` 迁移至 `流程n`，使用流程式生成代码，请参考默认示例构建流程
-   (可选) 如没有显示默认示例，请点击 `生成代码` - `恢复默认设置` 以恢复默认示例
